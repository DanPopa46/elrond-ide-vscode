Dependencies

Item Site Details
C buildchain LLVM

In order to build smart contracts to WASM bytecode, a subset of LLVM is necessary. More specifically, clang, llc and wasm-ld.

Rust buildchain rust-lang This is required in order to build smart contracts with Rust programming language.
Solidity buildchain Second State SOLL compiler, plus LLVM (8.0.0)

Second State - SOLL is a new compiler for generating Ewasm (Ethereum flavored WebAssembly) files from Solidity.

Elrond nodedebug elrond-go-node-debug

This variant of the Elrond node is required for debugging smart contracts. It enables a HTTP endpoint which is then used by the Visual Studio Code extension to deploy smart contracts and execute exported functions.
